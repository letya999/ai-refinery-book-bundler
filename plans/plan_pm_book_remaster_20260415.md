## Fix 1 — `scripts/chapter_processor.cjs`: chapterMap + lang

### 1a. Add glossary.html to chapterMap
In `navScript` template (around line 144), after the for-loop, add:
```
chapterMap['glossary.html'] = ${totalChapters} - 1;
```
This routes `href="glossary.html#wbs"` to chapter index = last chapter (glossary),
then the existing `anchorId` scroll logic scrolls to the correct anchor.

### 1b. Pass `lang` param to prepareChapter
- Add `langCode = 'ru'` parameter to `prepareChapter()` signature
- Replace hardcoded `'<html lang="en">'` (line 230) with `'<html lang="${langCode}">'`
- `bundle.cjs` already has `langCode` variable — pass it through `prepareChapter()`

## Fix 2 — `scripts/convert_pm_book.py`: Full SVG Enrichment

Replace the 2-SVG hardcoded approach with a chapter-semantic SVG library.
Each chapter gets domain-appropriate visuals using only CSS variables.

### SVG Rules (ALL must use CSS vars, not hardcoded hex):
- Nodes: `class="diag-node"` (= `fill: var(--panel2); stroke: var(--line)`)
- Text: `class="diag-text"` (= `fill: var(--txt)`)
- Accent node: `style="fill:var(--acc)"` + text `style="fill:var(--bg)"` (contrast)
- Bad/critical: `style="stroke:var(--bad)"` or `style="fill:var(--bad)"`
- Links: `class="diag-link"` (= `stroke: var(--line)`)

### Diagrams per chapter:

| Chapter | Content | Diagram Type | SVG Size |
|---------|---------|-------------|----------|
| Ch 3 (Подход) | Waterfall vs Agile | Translator + comparison blocks | CSS only |
| Ch 4 (Инициация) | Project charter process | BPMN-style flowchart SVG | 600x180 |
| Ch 5 (Планирование) | Planning phases | Timeline SVG (horizontal) | 700x120 |
| Ch 6 (Содержание) | WBS decomposition | Tree SVG | 600x220 |
| Ch 7 (Команда) | RACI matrix | HTML `<table class="matrix">` | — |
| Ch 8 (Время/Стоимость) | Gantt chart + Network | Gantt SVG (bars) + Critical Path SVG | 700x200 + 500x150 |
| Ch 9 (Ответственность) | RACI + Communication | RACI table + Radar SVG | 400x300 |
| Ch 10 (Риски) | Risk probability/impact matrix | 2×2 heatmap SVG | 400x300 |
| Ch 11 (Другие планы) | Change control flow | Flowchart SVG | 500x200 |
| Ch 12 (Выполнение) | Control cycle | Circular process SVG (4-step) | 400x300 |

### Gantt SVG spec (for Ch 8):
- X-axis: weeks/sprints (text labels)
- Y-axis: tasks (3-5 bars)
- Bars: `<rect>` with `fill:var(--acc)` and `fill:var(--acc2)` alternating
- Critical path bar: `fill:var(--bad)`
- All text: `class="diag-text"` (inherits `var(--txt)`)

### Risk Matrix SVG spec (for Ch 10):
- 3×3 grid of `<rect>` cells
- Low risk cells: `fill: rgba(135,239,201,.25)` (acc2 tinted)
- Medium: `fill: rgba(255,209,138,.25)` (warn tinted)
- High: `fill: rgba(255,100,100,.25)` (bad tinted)
- Labels: `class="diag-text"` with position
