Модуль 3: Агрегация и Группировка

Агрегатные функции позволяют вычислять итоговые значения для групп строк, превращая массивы данных в конкретные бизнес-показатели.

COUNT()

Считает количество строк. COUNT(*) — все строки, COUNT(col) — не NULL.

SUM()

Арифметическая сумма числовых значений в столбце.

AVG()

Среднее арифметическое (только для чисел).

MIN / MAX

Минимальное и максимальное значение (числа, даты, текст).

SELECT snum, COUNT(cnum), SUM(amt)
FROM Orders
GROUP BY snum;

AGGREGATION (Math)

GROUP BY разделяет таблицу на виртуальные группы по указанному столбцу.

Фильтрация групп: HAVING

В отличие от WHERE, который фильтрует строки до группировки, HAVING фильтрует уже готовые группы.

SELECT city, AVG(rating)
FROM Customers
GROUP BY city
HAVING AVG(rating) > 200;

Ограничения

  • В SELECT нельзя мешать обычные столбцы и агрегаты (без GROUP BY).
  • Агрегатные функции (кроме COUNT) игнорируют NULL.
  • HAVING работает медленнее WHERE, используйте его только для условий по агрегатам.
