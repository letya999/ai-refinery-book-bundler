Проектирование и Администрирование

Создание таблиц, управление индексами и обеспечение безопасности через систему привилегий. Основы архитектуры БД.

Связь объектов БД: Таблицы, Индексы и Представления
CREATE TABLE: Определение структуры

Команда DDL for создания физической структуры таблицы, определения типов данных и ограничений (Constraints).

INDEX: Ускорение поиска

Специальная структура данных, позволяющая СУБД находить строки без полного сканирования таблицы. Плата за скорость — замедление вставок.

VIEW: Виртуальные таблицы

Сохраненный запрос, который выглядит для пользователя как обычная таблица. Идеально для скрытия сложности или ограничения доступа.

Создание таблицы с ключом 
CREATE TABLE Salespeople (
  snum integer PRIMARY KEY,
  sname varchar(20) NOT NULL,
  city varchar(15),
  comm decimal
);

Предоставление разрешений пользователям на выполнение определенных действий (SELECT, UPDATE и т.д.).

REVOKE

Отозвание ранее выданных привилегий.
