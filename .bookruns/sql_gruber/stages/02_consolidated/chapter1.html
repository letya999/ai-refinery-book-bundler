SQL: Архитектура и Синтаксис

SQL — это декларативный язык для работы с реляционными данными. В отличие от императивных языков, здесь вы описываете что хотите получить, а не как это сделать.

DDL (Data Definition) DML (Data Manipulation) DCD (Data Control)

Анатомия Данных

Базовый синтаксис SELECT

SELECT [DISTINCT] { * | column_list } FROM table_name [WHERE condition] [GROUP BY columns] [HAVING condition] [ORDER BY columns [ASC|DESC]];

Типы данных

Числовые INTEGER (целые), DECIMAL (фиксированная точка), FLOAT (плавающая точка).
Символьные CHARACTER (фиксированная длина), VARCHAR (переменная длина).
Временные DATE (дата), TIME (время), TIMESTAMP (дата и время).

Создание таблиц (DDL)

CREATE TABLE table_name ( column1 data_type PRIMARY KEY, column2 data_type NOT NULL, column3 data_type REFERENCES other_table(id) );

Модификация данных (DML)

INSERT INSERT INTO table VALUES (...);
UPDATE UPDATE table SET col = val WHERE ...;
DELETE DELETE FROM table WHERE ...;
