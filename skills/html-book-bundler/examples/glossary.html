Glossary

Key concepts and definitions used throughout the book.

Minimum Viable Product (MVP)

A version of a new product which allows a team to collect the maximum amount of validated learning about customers with the least effort.

srcdoc (HTML Attribute)

An HTML attribute of the <iframe> element that specifies the HTML content of the page to show in the inline frame. Used by this bundler for high-performance offline chapter loading.

Search Index (SIDX)

A lightweight, bundle-time generated inverted index used for full-text search without external dependencies.
